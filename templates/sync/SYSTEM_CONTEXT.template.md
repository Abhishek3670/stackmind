## Shell Notes

### Windows (PowerShell)
- `&&` does NOT work — use `;` to chain commands
- Use `\` for path separators in commands
- Activate venv: `.\.venv\Scripts\Activate.ps1`

### Linux/macOS (Bash)
- `&&` works for command chaining
- Use `/` for path separators
- Activate venv: `source .venv/bin/activate`
