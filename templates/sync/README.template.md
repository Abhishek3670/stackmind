## 🚀 Agent Boot Sequence

```
0. READ  AGENTS.md (project root)
1. READ  runtime/boot/<agent>.boot.yaml
2. PEEK  TREE.yaml tree_version
3. CHECK PROTOCOL_DIGEST.hash
4. CHECK inbox (skip if unread=0)
5. CHECK decisions (skip if watermark current)
6. READ  assigned work orders
7. RESUME from next_action
```

---

## 🛑 Agent Shutdown Sequence

```
1. WRITE  draft snapshot → runtime/drafts/
2. WRITE  session report → outbox/
3. ARCHIVE inbox → inbox/<agent>/_read/
4. OUTPUT Handoff Report
5. COMMIT .sync repo
```
